## Accessing the TomoChain Assets Management

To access the TomoChain Assets Management, visit [tomoChain.vukhaihoan.com](https://tomoChain.vukhaihoan.com/)

## Development

### Install Dependencies

```bash
yarn
```

### Run

```bash
yarn start
```
